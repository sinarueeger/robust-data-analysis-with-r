1.3 Automation

Programming is essentially an automation of a process - for repeatable tasks (like adding two numbers) - scalable (many times repeating ) - complicated tasks (variable selection) - where precision is needed (adding two decimals) - reusing of frameworks (long code into functions) - reproduce it

A side effect of a certain automation also brings reproducibility.
